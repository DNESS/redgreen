RedGreen
========

A tiny game written for the "My First Raspberry Pi Game" series of videos:

http://www.artificialworlds.net/blog/category/my-first-raspberry-pi-game/

http://www.youtube.com/playlist?list=PLgyU3jNA6VjS3ij6ZXbb2x4GdEP3bAWzO

Enjoy!


Prerequisites
=============

On the Raspberry Pi with Raspian, should work out of the box.

On Ubuntu open a terminal and type:

sudo aptitude install python-pygame

It should be similar on other Linuxes, and should work on Windows and Mac too -
please let me know.


Running
=======

Open a terminal in the directory containing redgreen.py and type:

./redgreen.py

or

python redgreen.py
